O que é

HTML5(Hipertext Markup Language,versão 5) é uma linguagem para estruturação e apresentação de conteúdo para a world wide web e é uma tecnologia chave da Internet.

Essa é a quinta Versão da linguagem HTML.Esta nova versão traz consigo importantes mudanças quanto ao papel do HTML no mundo da web,através de novas funcionalidades como semântica e acessibilidade. Com novos recursos,antes só possiveis por meio de outras tecnologias.Sua essencia tem sido melhorar a linguagem com o suporte para as mais recentes multimídias,enquanto a mantém facilmente legível por seres humanos e consistentemente compreendida por computadores e outros dispositivos.Atualmente,está em fase de desenvolvimento e experimentação,porém diversos navegadores implementam algumas de suas funcionalidades.

O que há de novo?

Novos elementos Novos atributos Suporte Total para CSS3 Suporte para audio e vídeo gráficos em 2D e 3D Armazenamento local Aplicações web

Alguns dos mais interessantes recursos em HTML5 são:

O elemento <canvas>para desenho 2D;
Os elementos de <video>e <audio> para a excução de media;
Suporte de armazenamento local;
Novos elementos de especificação de conteúdo,como <article><footer><header><nav><section>
Novos controles de formulários,como calendar,date,email,url,e search.
